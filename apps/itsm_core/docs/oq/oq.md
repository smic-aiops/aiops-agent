# OQ（運用適格性確認）: ITSM Core（SoR）

## 目的

ITSM Core（SoR）はサブアプリ構成のため、OQ は **サブアプリ単位**で整備する。
本書（`apps/itsm_core/docs/oq/oq.md`）は OQ の入口として、正の OQ ドキュメントを案内する。

## OQ ドキュメント（正）

- SoR コア Webhooks（監査イベント投入 / AIOps 互換 Webhook / 同期）:
  - `apps/itsm_core/sor_webhooks/docs/oq/oq.md`
- GitLab backfill（テスト投入）:
  - `apps/itsm_core/gitlab_backfill_to_sor/docs/oq/oq.md`
- その他のサブアプリ:
  - `apps/itsm_core/<app>/docs/oq/oq.md`
