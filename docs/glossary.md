# 用語集（本リポジトリでの読み替え）

本書は、README で使用する用語を「最小で試す」目的で読み替えたものです。

| 用語 | 何か？ | 主目的 |
|---|---|---|
| **ITIL** | ベストプラクティス | IT サービスの価値最大化 |
| **ITSM** | 運用体系 | IT サービスを安定運用 |
| **ITOM** | 運用技術領域 | IT 基盤を見える化・自動化 |
| **ITAM** | 資産管理領域 | IT 資産の最適化・コスト管理 |
| **SecOps** | セキュリティ運用 | 脅威の検知・対応 |
| **IRM（GRC）** | ガバナンス管理 | リスク・統制・監査 |
| **SPM / PPM** | 戦略・投資管理 | IT 投資の最適化 |
| **ESM** | 全社展開概念 | ITSM を全社業務へ拡張 |
| **DevOps** | 開発と運用の協働思想 | 変更の高速化と品質・安定性の両立 |
| **GAMP® 5** | CSV の実践ガイドライン | システムの妥当性を効率的に保証 |
| **Part 11** | 電子記録・電子署名規制 | 電子データの信頼性・完全性確保 |
