本書は「なぜこの基盤が必要か」を説明するための文章です。監査向けの構成説明や検証導線は、ルートの `README.md` と `docs/README.md` を正とします。

## 変えられないを変革する

多くの企業では、ツールが分断されており、データや業務が十分につながっていません。IT 運用は属人化しがちで、改善や DX に手が回らない状況が続いています。インシデント対応に追われることで、将来への投資が後回しになり、AI や自動化に関心はあっても、基盤の見直しに着手できないという課題が存在します。

## AIOps Agent サービス管理基盤とは

AIOps Agent サービス管理基盤は、DX・業務自動化・一元データ管理を実現するための「基盤（＝土台）」を提供するサービスです。既存の IT サービスや社内リソースを活かしながら、不足している部分を補完し、「運用の安定化」「追補型の基盤整備」「自動化・AI 活用」を並行して進めていきます。

段階的に将来の DX へと進化させていく点が大きな特長です。

## AI ワークフロー x AIOps エージェント｜業務に合わせて育てる AI

業務内容に応じた AI ワークフロー（主にサービスリクエスト向け）を構築することが可能です。作成したワークフローはカタログとして自動生成され、一覧で利用できます。

AIOps エージェントは、サービス運用者が業務に合わせてカスタマイズ可能です。プロンプトやポリシー（ルール）を設定・変更することで、動作を柔軟に調整できます。本基盤では、AI を用いたワークフローや AI エージェントの設計・育成は現場主導で進められます。

※ AI モデルを稼働させるための API キー（OpenAI 互換 API Key 等）は本リポジトリでは提供していません。利用者側でご用意ください。

## AIOps Agent サービス管理基盤の価値

運用・改善・基盤整備を止めずに継続できます。DX を一括導入ではなく、「継続的な進化」として実現できる点が大きな価値です。将来の拡張や高度化を前提とした設計により、長期的な DX 推進を支援します。

## AIOps Agent サービス管理基盤が目指す未来

AI と自動化が日常業務に自然に組み込まれる世界を目指します。現場主導で改善が回る組織を実現し、継続的に進化する DX 基盤を確立します。

初期費用を抑え、無駄なコストを発生させることなく、「手軽に、安く、最大の成果」を実現します。

## 選ばれる理由

- 既存システムを活かし、無理なく DX を推進
- 強固なセキュリティとデータレジデンシー（国産 AI モデル設定も可能）
- AI エージェントを DX の中核に据えた基盤
