# 行動規範（Code of Conduct）

本プロジェクトでは、参加者が安全で生産的に協力できる場を作るため、相互尊重を前提とします。

## 期待する行動

- 相手の立場を尊重し、建設的なフィードバックを行う
- 事実・再現手順・根拠リンクを重視し、個人攻撃をしない
- 機微情報（資格情報、個人情報、顧客情報、内部 URL 等）を公開しない

## 許容されない行動

- ハラスメント、差別、脅迫、侮辱、個人攻撃
- 秘密情報の暴露、無断のログ/証跡共有
- 迷惑行為（スパム、過度な煽り、故意の妨害）
